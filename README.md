## 实时抓取中国上市互联网公司市值和其排名

- **安装**
```bash
#克隆本项目
git clone https://github.com/zhyee/Ranking-of-Chinese-Internet-Corp.git

#composer安装
cd Ranking-of-Chinese-Internet-Corp
composer install
```

- **使用**
```bash
php public/index.php
```
